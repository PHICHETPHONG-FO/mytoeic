# 🚀 TOEIC 60 วัน พิชิตเป้า — 2-Month TOEIC Quest

แผนติว TOEIC 2 เดือน แบบ **gamified** เปิดในเบราว์เซอร์ได้ทันที ครบทั้ง 4 ทักษะ (ฟัง–อ่าน–พูด–เขียน) พร้อมระบบเก็บ XP เลื่อนแรงก์ เหรียญตรา และ checklist ที่บันทึกความคืบหน้าอัตโนมัติ เรียนสนุก ไม่เบื่อ และเห็นพัฒนาการชัดเจน

> ไฟล์เดียวจบ ไม่ต้องติดตั้งอะไร ไม่ต้องต่อเน็ตก็ใช้ได้ (ยกเว้นฟอนต์)

---

## ✨ จุดเด่น (Features)

- **40 วันเรียน** — 8 สัปดาห์ × 5 วัน (หยุดเสาร์–อาทิตย์) วันละ 2–4 ชม.
- **ครบ 4 ทักษะ + Speaking & Writing** — ตรงตามโครงสร้างข้อสอบจริงทั้ง 7 พาร์ท
- **สองภาษาคู่กัน** — ไทยเป็นหลัก มีอังกฤษกำกับ ช่วยซึมซับศัพท์ไปในตัว
- **Day 1 เป็น Diagnostic Test** — วัดระดับก่อนเริ่ม แล้วปรับความยากเองได้ทุกระดับ
- **สอบซ้อมจริง 2 ครั้ง** — วันที่ 20 (กลางคอร์ส) และวันที่ 36 เทียบพัฒนาการ
- **ระบบเกมกันเบื่อ** — เก็บ XP, เลื่อน 8 แรงก์ (มือใหม่ → ตำนาน 900+), สตรีครายวัน 🔥, เหรียญตรา 8 แบบ
- **เกมท้ายบททุกวัน** + คลังเกม/แอปแยกตามทักษะ (Quizlet, Kahoot, ELSA Speak, Grammarly, BBC Learning English ฯลฯ)
- **Checklist บันทึกอัตโนมัติ** — ใช้ `localStorage` ปิดแล้วเปิดใหม่ความคืบหน้าไม่หาย
- **เช็กลิสต์วันสอบจริง** — เตรียมตัวก่อนเข้าห้องสอบ

## 🚀 วิธีใช้งาน (How to use)

**วิธีที่ 1 — เปิดตรงๆ**

1. ดาวน์โหลดหรือ clone repo นี้
2. ดับเบิลคลิกไฟล์ `index.html`
3. เริ่มที่ Day 1 (Diagnostic) แล้วติ๊ก checklist ไปเรื่อยๆ

```bash
git clone https://github.com/phichetphong-fo/mytoeic.git
cd mytoeic
# แล้วเปิดไฟล์ index.html ในเบราว์เซอร์
```

**วิธีที่ 2 — เปิดบนมือถือ**

ส่งไฟล์ `.html` เข้ามือถือแล้วเปิดด้วยเบราว์เซอร์ได้เลย (รองรับหน้าจอเล็ก)

> 💡 ความคืบหน้าถูกเก็บใน `localStorage` ของเบราว์เซอร์นั้นๆ — ใช้เครื่อง/เบราว์เซอร์เดิมเพื่อให้ข้อมูลต่อเนื่อง และอย่าล้าง site data

| สัปดาห์ | ธีม | โฟกัส |
|:---:|---|---|

## 🎮 ระบบเกม (Gamification)

- **XP** — ทำภารกิจสำเร็จ +10 XP ต่อข้อ, เรียนจบครบทั้งวัน โบนัส +40 XP
- **8 แรงก์** — 🐣 มือใหม่หัดขับ → 📚 นักเรียนขยัน → ⚔️ จอมยุทธ์ไวยากรณ์ → 🎧 เซียนฟังจับใจความ → ⚡ นักอ่านสายฟ้า → 👑 ราชานักพูด-นักเขียน → 🧙 ปรมาจารย์ TOEIC → 🏆 ตำนาน 900+
- **สตรีค 🔥** — นับวันที่เรียนต่อเนื่อง
- **เหรียญตรา** — ปลดล็อกเมื่อถึงหมุดหมาย (จบสัปดาห์แรก, ครึ่งทาง, ผ่านสอบซ้อม, จบหลักสูตร ฯลฯ)

## 🛠️ เทคโนโลยี (Tech Stack)

- HTML + CSS + JavaScript ล้วน (Vanilla) — **ไฟล์เดียว ไม่มี dependency ไม่ต้อง build**
- `localStorage` สำหรับบันทึกความคืบหน้า
- Google Fonts: `Kanit`, `Sarabun` (โหลดผ่าน CDN — ส่วนอื่นทำงานออฟไลน์ได้)

## ⚙️ การปรับแต่ง (Customization)

เนื้อหาหลักสูตรอยู่ในตัวแปร `WEEKS` ในแท็ก `<script>` แก้ได้ตรงๆ:

```js
const WEEKS = [
  { n:1, color:"var(--teal)", th:"...", en:"...", days:[
    { id:"d1", day:1, th:"ชื่อวัน", en:"Day title",
      skills:["M","RV"], time:"2.5 ชม.",
      goal:"เป้าหมายของวัน",
      schedule:[ {tm:"20 นาที", ac:"กิจกรรม", acEn:"activity"} ],
      tasks:["ภารกิจ 1", "ภารกิจ 2"],
      game:{ name:"ชื่อเกม", desc:"คำอธิบาย", url:"example.com" },
      tip:"เคล็ดลับครู"
    }
  ]}
];
```

- **เพิ่ม/ลดภารกิจ** — แก้ใน `tasks` (XP คำนวณอัตโนมัติ)
- **ปรับเกณฑ์แรงก์** — แก้ค่า `min` ใน `RANKS` (ค่าสูงสุดต้องไม่เกิน XP รวมที่ทำได้จริง)
- **เปลี่ยนสี/ธีม** — แก้ CSS variables ในส่วน `:root`
